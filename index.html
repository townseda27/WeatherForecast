Overview

This site was created by Daniel Townsend, a computer science major at Miami University. This home page describes the project and how it was built, and the weather page provides users the ability to search for a location and recieve a five day weather forecast. The project is made up of two web pages:

  • Home page
  • Weather page
About the system
Home page

The home page (current page) is a simple HTML page made with CSS and Bootstrap. This page simply describes the project, it's components and the developer. Bootstrap was used to align the text, image and the cards as well as giving the page responsiveness as you shrink or enlarge the page.

Weather page

The weather page uses HTML, CSS, Bootstrap, JS, AJAX, and JQuery to style and format the page as well as allowing for API calls to be made and processed. Users can enter keywords to search for via the text box and the javascript logic will begin the process of calling APIs. First, the TomTom API is called to fetch location data which is then fed into the OpenWeatherMap API to retrieve forecast data for the specified location.
